# 🌦️ Weather App

A simple and elegant weather web application that automatically detects your location or lets you search any city worldwide — showing temperature, humidity, wind speed, and conditions with a dynamic background that changes to match the weather.

## ✨ Features

- 🌍 Auto-detects your location on load
- 🔍 Search by any city name
- 🌤️ Displays:
  - Temperature
  - Feels-like temperature
  - Humidity
  - Wind speed
  - Weather description & icon
- 🎨 Dynamic background that changes based on current weather conditions

## ⚙️ Setup Instructions

1. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
2. Replace the placeholder API key in `script.js`:

```js
const apiKey = "YOUR_API_KEY";
```

3. Open `index.html` in your browser — that's it!
