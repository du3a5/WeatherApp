# 🌦️ Weather App

A simple and elegant weather web application built with **JavaScript**, **HTML**, and **CSS**.  
The app automatically detects your location or allows you to search for any city worldwide.  
It displays temperature, feels-like temperature, humidity, wind speed, and weather conditions,  
with a dynamic background that changes based on the current weather.

----

## ✨ Features
- 🌍 Auto-detects your location on load  
- 🔍 Search by any city name  
- 🌤️ Shows:
  - Temperature  
  - Feels like temperature  
  - Humidity  
  - Wind speed  
  - Weather description & icon  
- 🎨 Changes background automatically based on weather conditions
